# usd-jpy-converter
Rustで作成したUSD↔JPYの為替レート変換のCLIアプリ

## 概要

**usd-jpy-converter** は、最新のドル円為替レートを自動で取得し、  
- ドル→円
- 円→ドル
の相互変換結果を即時表示できるRust製CLIアプリです。

コマンド例:  
```bash
cmd -d 599     # 599ドルを現在レートで日本円に換算
cmd -y 1000    # 1000円を現在レートで米ドルに換算
```
実行結果には  
- 現在のUSD/JPYレート
- 換算結果
- レート取得時刻  
を同時に表示します。

## 主な特徴

- コマンドラインでシンプルに為替計算
- APIからリアルタイムレート取得
- ドル円どちらも相互変換可能

## 使い方

任意ディレクトリでビルド後、バイナリにPATHを通すことで  
どこからでも `cmd` コマンド（または任意名称）として利用できます。
